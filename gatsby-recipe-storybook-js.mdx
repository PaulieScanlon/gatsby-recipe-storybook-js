## You're all done!

Now you just need to add some scripts to package.json

"storybook": "NODE_ENV=production start-storybook -s ./public -p 6006 --ci",

"build-storybook": " NODE_ENV=production build-storybook",

## Oh, and before I forget!

Before your run storybook make sure you run gatsby develop

This is so GraphQL data is available in Storybook
